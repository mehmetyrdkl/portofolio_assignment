Fonts and UI components

For this assignment we had to create a font hierarchy with one or two fonts. I verified my hierarchy using type-scale, and searched for fonts on Google Fonts, but also adjusted the margins, paddings and line heights.

Using a serif font as the display font, set in my variables, aswell as a sans serif font for my paragraph font. Of course the standard sized from type-scale were not enough, so I had to change these accordingly to the specific sections I was working on at that moment.

Besides focusing on fonts, we also had to improve the UI components, therefore, the cards that I created for this grayscale assignment are using CSS flex, the borders on them are subtle, as well as the borders of the buttons, adding a hint of darker grey on the button on which we want the user to focus on.

The link for the solution can be found in the breadcrumbs - Fonts and UI components.
