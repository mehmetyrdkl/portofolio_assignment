Working with colors and animations

For this assignment we had to apply a color paletter to a mockup landing page design and then add animations. However this was split in three parts, first apply the color paletter, then create a dark theme for the solution and then add animations.

I first proceeded with using Huemint, a solution that generated colors on a mockup page, showing the contrast of these, adding accent colors as necessary or as pleased. The next step was to validate these colors using Adobe's Contrast Analyzer.

Once I picked the colors and validated them, I then added them to my root variable and styled the mockup landing page accordingly. Second, I implemented the dark theme using a toggle button, that on click, my root color variables would change to the dark theme ones.

Furthermore, for animations, I used Motion One as my animation library, since we have already used it in a copuple of exercises in class. I added a simple animation that when a section is in view it would make the elements with the class "fade" appear from left to right with easing.

The link for the solution can be found in the breadcrumbs - Working with colors and animations.
